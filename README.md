# florentina

## Database structure

### Flower
* description
* info
* name
* sizes
* img
* catID
* createdAt
* updatedAt

## Install
    1. Install django 
    ```sudo pip install django```
    2. Clone the repo 
    ```git clone https://github.com/maksadbek/florentina```
    3. Make migrations
    ```python manage.py makemigrations```
    ```python manage.py migrate```
    4. Create superuser
    ```python manage.py createsuperuser```
    5. Run server
    ```python manage.py runserver```
